Conditional draft

A Public Normal-Form Framework for Entropy Bookkeeping in Sunflower Bounds

Yuli Li, Jingping Yang, Renhao Zhang

This is a public draft and reproducibility package for a conditional framework. It does not claim an unconditional proof of the Erdos-Rado sunflower conjecture. The main theorem is an interface reduction: if the public transition-system certificates specified in the paper are all verified, the framework would imply a constant-base bound for balanced two-collision codes.

What To Inspect First

The candidate alphabet-free mechanism is the fixed labelled hash window: low-collision coordinates are placed into a bounded public window without exposing alphabet values, and repeated use is charged to fresh, old, and scale potentials.
